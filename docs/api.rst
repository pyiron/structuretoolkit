Interface
=========

Documentation of the classes and functions defined in the :code:`structuretoolkit` package. Every function
takes an :code:`ase.atoms.Atoms` object as input and, where applicable, returns one again, so
:code:`structuretoolkit` can be combined freely with the rest of the `ASE <https://wiki.fysik.dtu.dk/ase/>`_
ecosystem.
